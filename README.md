# 🐱🐶 喵汪图说生成器

> 把小说变成可爱的猫猫狗狗图说 📖→🐾

## ✨ 功能特点

- 📝 **粘贴小说** - 支持任意长度文本
- 🎭 **5种角色** - 傲娇猫咪、阳光狗狗、文艺猫猫、憨憨狗子、猫狗组合
- 🎨 **自动生成** - 智能分段，生成1-8张图说卡片
- 🖼️ **随机宠物图** - 每张卡片配备不同的猫咪/狗狗照片
- 💾 **导出功能** - 复制文字、下载PNG图片
- 📱 **全平台响应** - 手机、平板、电脑完美适配
- ⚡ **无需后端** - 纯前端应用，完全本地运行

## 🎭 5种宠物角色

### 1. 😼 傲娇猫咪
- **特点**：高冷、内心柔软
- **语气**：哼、喵~、本喵
- **适合**：文艺、暗黑系小说

### 2. 🐕 阳光狗狗
- **特点**：热情、开朗、小太阳
- **语气**：汪汪、超级、太棒了
- **适合**：温暖、治愈系小说

### 3. 😻 文艺猫猫
- **特点**：温柔、细腻、爱生活
- **语气**：呢~、嗯呢、愿温柔相伴
- **适合**：青春、唯美系小说

### 4. 🐶 憨憨狗子
- **特点**：天真、单纯、傻乎乎
- **语气**：嗷~、修狗、好像有点明白
- **适合**：喜剧、温馨系小说

### 5. 🐱🐶 猫狗组合
- **特点**：互动、对话、一唱一和
- **语气**：🐱：... 喵~、🐶：... 汪！
- **适合**：所有风格（双角色对话）

## 💡 使用技巧

1. **快速生成**：`Ctrl + Enter`（Cmd + Enter on Mac）
2. **导出图片**：点击"保存全部卡片"自动下载PNG
3. **分享文字**：点击"复制全部文字"粘贴到社交媒体
4. **多次生成**：同一文本可用不同角色多次生成不同效果

---

## 🛠️ 技术栈

- **前端框架**：纯 HTML5 + CSS3 + Vanilla JavaScript
- **样式系统**：CSS变量 + Grid + Flexbox
- **动画**：CSS Keyframes + Transform
- **图片源**：
  - 猫咪：[cataas.com](https://cataas.com)
  - 狗狗：[placedog.net](https://placedog.net)
- **导出工具**：[html2canvas](https://html2canvas.hertzen.com/)

## 🌐 部署方式

### GitHub Pages（推荐）
自动部署在：`https://800coke.github.io/twzh/`

**启用步骤**：
1. 进入仓库 Settings
2. 找到 "Pages" 选项
3. Source 设置为 "Deploy from a branch"
4. Branch 选择 "main"
5. 点击 Save

✅ 大约 1-2 分钟后即可访问

### Vercel 快速部署
[![Deploy with Vercel](https://vercel.com/button)](https://vercel.com/new/clone?repository-url=https%3A%2F%2Fgithub.com%2F800Coke%2Ftwzh)

1. 进入 [Netlify](https://netlify.com)
2. 连接 GitHub
3. 选择此仓库自动部署
